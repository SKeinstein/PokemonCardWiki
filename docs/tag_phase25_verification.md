## 5. ダメカン操作サブタグ検証

### 5-1. サブタグ件数

| サブタグ | 件数 | 設計通りか |

**3サブタグ合計:** 116 = ダメカン操作親タグ件数と一致。ただし1カードが複数サブタグを持つ可能性はないか確認済み（移動型は のせる型でも反射型でもないため排他）。

### 5-2. 反射型とカウンター効果の重複

ダメカン操作>反射 33件全件が `カウンター効果` を同時保持 ← 設計通り。

代表カード: テツノイワオex / テツノコウベ / ミミズズex / ハギギシリ / マラカッチ / スパイクエネルギー / デラックスボム / ブリガロン / メガヤドランex / メガズルズキンex 他

### 5-3. ベンチ狙撃>ダメカン型との境界（※2 補足）

ドラパルトex のテキスト「ダメカン6個を、相手のベンチポケモンに好きなようにのせる」は、ダメカン操作>のせるの検出パターン `/ダメカンを.{1,30}のせる/`（`ダメカンを`直後マッチ）に非該当（テキスト中は`ダメカン6個を`）。

このため、ドラパルトexは `ベンチ狙撃>ダメカン型`のみ保持し `ダメカン操作`系は付与されない。一方、ヨノワール/サマヨール/ペロッパフは「ダメカンを2個のせる」形式で `ダメカン操作>のせる` + `ベンチ狙撃>ダメカン型` 両方を保持。この境界は設計上の意図的差異（ベンチ狙撃メインの攻撃型 vs ダメカン設置型の区別）。
