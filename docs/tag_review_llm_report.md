### Root Cause — Broken Regex in §3-4

The existing pattern `/相手の手札の枚数.{1,5}[×x]\d+ダメージ/` never matched because:

- Actual card text: `相手の手札の枚数×30ダメージ`
- Pattern requires `.{1,5}` (1–5 any chars) **followed by** `[×x]`
- In practice `.{1,5}` consumes the `×` character, then `[×x]` has nothing to match
- The pattern silently failed for all `枚数×Nダメージ` forms

Additionally, `/手札.{1,5}同じ枚数/` was too broad — it matched hand-parity conditions unrelated to damage (テツノイワオ's attack validity and オンバーン's energy cost reduction).

### Fix Applied to `scripts/tag_cards.mjs` (§3-4)

```javascript
// BEFORE (broken):
// §3-4  手札枚数参照
if (
  /相手の手札の枚数.{1,5}[×x]\d+ダメージ/.test(all) ||  // never matched
  /手札の枚数ぶん引く/.test(all) ||
  /手札.{1,5}同じ枚数/.test(all)                        // too broad
) {
  add('手札干渉', '手札干渉>手札参照');
}

// AFTER (fixed):
// §3-4  手札枚数参照（ダメージ計算に手札枚数を使うカード）
// Matches: 相手の手札の枚数×Nダメージ (e.g. シャンデラ, メガユキメノコex)
//          自分の手札と相手の手札が同じ枚数なら、X ダメージ追加 (e.g. ゴチルゼル シンクロショット)
// Does NOT match: hand-count as attack-validity condition (テツノイワオ アジャストホーン)
//                 hand-count as energy-cost condition (オンバーン チューニングエコー)
if (
  /手札の枚数[×x]\d+ダメージ/.test(all) ||
  /手札.{1,10}同じ枚数.{1,100}ダメージ追加/.test(all)
) {
  add('手札干渉', '手札干渉>手札参照');
}
```
