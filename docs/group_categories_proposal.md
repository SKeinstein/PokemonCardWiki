## 0. ルールガイド上の重要な機械的区別

サブグループ分類の根拠となるルール区別を先に整理する。

### ワザのダメージ vs ダメカンを乗せる（C-07 / B-08）

| 項目 | ワザのダメージ（B-08） | ダメカンを乗せる効果（C-07） |
|---|---|---|
| 処理手順 | ダメージ計算手順1〜6 | ワザ手順5（ダメージ以外の効果）として処理 |
| 弱点・抵抗力 | バトル場は適用あり。ベンチは「計算しない」旨が明記 | **常に適用なし** |
| ダメージ軽減（−●●） | 適用される | **適用されない** |
| 「ワザのダメージを受けない」 | 防ぐことができる | **防ぐことができない** |
| 「ワザの効果を受けない」 | ダメージは通る | ダメカン設置を防ぐことができる |
| テキスト識別子 | `○○ダメージ` ＋ `〔ベンチは弱点・抵抗力を計算しない。〕` | `ダメカンを●個のせる`（ブラケットなし） |

→ **ベンチ狙撃**は §4-1（ワザのダメージ型）と §4-2（ダメカン直置き型）に分類する。

### にげる vs 入れ替え効果（C-03）

| 項目 | にげる（A-03） | 入れ替え効果（C-03） |
|---|---|---|
| エネルギーコスト | 必要（トラッシュ） | **不要** |
| ねむり・マヒ下での使用 | **不可** | 可 |
| にげられない効果下での使用 | 不可 | **可** |
| 特殊状態のリセット | リセットされる | リセットされる |

→ **ポケモン入れ替え**（§1）と **にげるコスト**（§1-3）は機能が異なる別サブグループ。

### 相手入れ替えの方向性（C-04 vs C-05）

- **C-04型**（相手バトル→ベンチ、次のポケモンは相手が選ぶ）: 効果はバトルポケモンが受ける。バトルポケモンが「ワザの効果を受けない」状態なら不発。
- **C-05型**（相手ベンチ→バトル、自分が指定）: 効果はベンチポケモンが受ける。指定ベンチポケモンが「ワザの効果を受けない」状態なら不発（既存バトルポケモンは通常退場）。

→ §2 の2サブグループ分割の根拠。

### ポケモンチェックのタイミング（F章）

各番の終わりに発生。処理順：**どく → やけど → ねむり → マヒ → 特性/スタジアム効果 → きぜつ確認**。
次の番を行うプレイヤーが効果の処理順を決定可能（F章-2）。

- **どく**: ポケモンチェック時ダメカン1個（特性等で変化あり）
- **やけど**: ポケモンチェック時ダメカン2個 ＋ コイン表で状態解除
- **ねむり**: ポケモンチェック時コイン表で回復。攻撃・逃げ不可
- **マヒ**: 次の自分の番の終わりのポケモンチェック後に自動解除。攻撃・逃げ不可
- **こんらん**: ポケモンチェックでは処理されない。攻撃宣言時コイン判定（ウラで自分に30ダメカン）

→ §8 の「持続ダメカン型（どく/やけど）」と「行動制限型（ねむり/マヒ/こんらん）」のサブグループ分割の根拠。

### エネルギー付加の種別（C-09 vs C-10）

- **C-09（新規付加）**: 山札・手札・トラッシュからポケモンへ。通常の手張り（1ターン1枚）と独立して行える。
- **C-10（つけ替え）**: 場のポケモン間での移動。エネルギー枚数は増減しない。
- 「エネルギーをつけるたびに」トリガーする効果（E-07）は、C-09による複数付加でも1枚ごとに発動する。

→ §5（エネルギー加速・C-09）と §B-5（エネルギー移動・C-10）を別カテゴリとして分離。
